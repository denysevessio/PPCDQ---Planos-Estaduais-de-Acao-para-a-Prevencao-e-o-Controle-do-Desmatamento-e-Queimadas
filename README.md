# PPCDQ – Indicadores Ambientais

Este repositório documenta o processo de estruturação, padronização e qualificação de indicadores ambientais utilizados no monitoramento do PPCDQ.

Por se tratar de um projeto institucional em andamento, não são disponibilizados dados reais ou dashboards. O foco aqui é o processo de trabalho, os aprendizados técnicos e as decisões adotadas.
