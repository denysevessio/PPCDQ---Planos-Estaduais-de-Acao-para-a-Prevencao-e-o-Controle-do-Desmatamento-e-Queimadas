# PPCDQ – Indicadores Ambientais

> Status: projeto em andamento

Este repositório documenta o processo de estruturação, padronização e qualificação de indicadores ambientais utilizados nos Planos Estaduais de Ação para a Prevenção e o Controle do Desmatamento e Queimadas (PPCDQ).

Por se tratar de um projeto institucional em andamento, não são disponibilizados dados reais ou dashboards. O foco está no processo de trabalho, organização das bases e aprendizados técnicos.

## Escopo do trabalho
- Coleta e consolidação de dados provenientes de diferentes órgãos estaduais  
- Padronização de indicadores e estruturas de bases  
- Análise exploratória para verificação de consistência e rastreabilidade  
- Organização dos dados para uso analítico em Power BI  
- Apoio técnico à evolução de dashboards e fluxos de atualização  

## Ferramentas e técnicas
Power BI • SQL • Python (pandas) • ETL • Modelagem de dados • Análise exploratória
